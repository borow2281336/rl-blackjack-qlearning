# RL Blackjack Homework

- Среда: `Blackjack-v1` (Gymnasium)
- Алгоритмы: табличный Q-learning + ε-жадная стратегия (fixed & decaying ε)
- Ноутбук: `blackjack_rl.ipynb` содержит весь код и отчёт

Запуск:
1. Открыть ноутбук в Google Colab или Jupyter.
2. Установить зависимости (`gymnasium`, `numpy`, `matplotlib`).
3. Заполнить все ячейки сверху вниз.
